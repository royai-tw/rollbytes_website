# RollBytes 官方網站

RollBytes 官方網站專案。

## 技術棧

- **框架**：Next.js 14 (App Router)
- **樣式**：Tailwind CSS
- **語言**：TypeScript
- **部署**：Vercel / Firebase Hosting

## 開發指令

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 建置生產版本
npm run build
```

## 專案結構

```
rollbytes_website/
├── src/
│   ├── app/           # Next.js App Router 頁面
│   ├── components/    # 共用元件
│   └── styles/        # 全域樣式
├── public/            # 靜態資源
└── package.json
```
